# Hospital Management System Frontend

This is the React frontend for the Hospital Management System.

Main project documentation is available in the root README:

```text
../README.md
```

## Local Commands

```bash
npm install
npm start
npm run build
```

## API Configuration

Create `hms/.env` when you need to override the API URL:

```env
REACT_APP_API_URL=https://hospital-management-system-y3xo.onrender.com/api
```

The frontend currently defaults to the deployed Render backend when this variable is not set.
